# Movie Review WebApp

A browser-based movie review platform built with PHP, MySQL, and Bootstrap 5.

## Features
- Browse all movies with posters and descriptions
- Filter movies by category
- Register and login system
- Add reviews with ratings (1-5 stars)
- View latest reviews
- Responsive design for desktop and mobile

## Technologies
- PHP
- MySQL
- HTML, CSS, JavaScript
- Bootstrap 5

## How to Run
1. Clone or download this repository
2. Place the project in your web server directory (e.g., `htdocs` in XAMPP)
3. Create a MySQL database and import the provided SQL tables (`users`, `movies`, `categories`, `reviews`)
4. Adjust database connection in `functions.php`
5. Open `index.php` in your browser to start using the app
